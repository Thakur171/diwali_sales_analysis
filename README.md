# 🪔 Diwali Sales Analysis

## 📌 Project Overview
This project analyzes Diwali sales data to understand customer behavior and sales trends.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Analysis
- Gender vs Sales
- Age Group vs Sales
- State-wise Sales
- Product Category Analysis

## 🔍 Insights
- Age group 26-35 has highest purchasing power
- Females contribute more to sales
- Certain states show higher demand

## 📁 Dataset
Diwali Sales Dataset (CSV)

## 🚀 Conclusion
This analysis helps businesses understand customer trends and improve marketing strategies.
